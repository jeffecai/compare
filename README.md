# 图片对比工具

一个简单的图片交替对比工具，支持选择两张图片交替显示，方便进行视觉对比，并可导出为 GIF 动图。

## 安装依赖

```bash
pip install PyQt6 Pillow
```

## 运行

```bash
python image_compare.py
```

## 使用说明

1. **选择图片** — 点击「选择图片 A」和「选择图片 B」分别加载两张待对比的图片
2. **设置参数**
   - **交替频率**：两张图片切换的时间间隔（0.1 ~ 10.0 秒，默认 0.5 秒）
   - **交替次数**：自动切换的总轮数（1 ~ 999 次，默认 10 次）
3. **开始对比** — 点击「▶ 开始对比」，预览区会按设定频率自动交替显示两张图片
4. **停止对比** — 点击「⏹ 停止」可随时中断
5. **保存 GIF** — 点击「💾 保存 GIF」选择输出目录，将交替动画导出为 `compare.gif`

## 支持的图片格式

PNG、JPG/JPEG、BMP、TIFF、WebP
